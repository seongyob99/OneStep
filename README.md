# OneStep
일상 속 작은 습관을 쉽고 체계적으로 관리하는 사이트!

## 기술 스택
### Front-end
- 프레임워크 및 라이브러리 : React, Axios
- 언어 : SCSS, JavaScript
### Back-end
- 프레임워크 및 라이브러리: SpringBoot, Spring Data JPA, QueryDSL
- DBMS : MariaDB
- 언어 : Java(17)
### Tools
- IDE : IntelliJ, VSCode
- Version Control : Git, Github
